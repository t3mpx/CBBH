## Ffuf commands
| Command                                                                                                                                                                                                                                          | Description              |
| ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------ | ------------------------ |
| `ffuf -u http://url.com/FUZZ -c -w /usr/share/seclists/Discovery/Web-Content/directory-list-2.3-medium.txt:FUZZ -ic -sa -mc all -fc 404 -o dir_fuzz.txt`                                                                                         | Directory Fuzzing        |
| `ffuf -u http://url.com/indexFUZZ -c -w /usr/share/seclists/Discovery/Web-Content/web-extensions.txt:FUZZ -ic -sa -mc all -fc 404 -o ext_fuzz.txt`                                                                                               | Extension Fuzzing        |
| `ffuf -u http://url.com/blog/FUZZ.php -c -w /usr/share/seclists/Discovery/Web-Content/directory-list-2.3-medium.txt:FUZZ -ic -sa -mc all -fc 404 -o page_fuzz.txt`                                                                               | Page Fuzzing             |
| `ffuf -u http://url.comT/FUZZ -c -w /usr/share/seclists/Discovery/Web-Content/directory-list-2.3-medium.txt:FUZZ -recursion -recursion-depth 1 -e .php -v -ic -sa -mc all -fc 404 -o recursive_fuzz.txt`                                         | Recursive Fuzzing        |
| `ffuf -u https://FUZZ.url.com/ -c -w /usr/share/seclists/Discovery/DNS/subdomains-top1million-5000.txt:FUZZ -ic -sa -mc all -fc 404 -o subdomain_fuzz.txt`                                                                                       | Sub-domain Fuzzing       |
| `ffuf -u http://url.com/ -c -w /usr/share/seclists/Discovery/DNS/subdomains-top1million-5000.txt:FUZZ -H 'Host: FUZZ.url.com' -ic -sa -mc all -fc 404 -o vhost_fuzz.txt`                                                                         | VHost Fuzzing            |
| `ffuf -u http://url.com/admin/admin.php?FUZZ=key -c -w /usr/share/seclists/Discovery/Web-Content/burp-parameter-names.txt:FUZZ -ic -sa -mc all -fc 404 -o param_get_fuzz.txt`                                                                    | Parameter Fuzzing - GET  |
| `ffuf -u http://url.com/admin/admin.php -c -w /usr/share/seclists/Discovery/Web-Content/burp-parameter-names.txt:FUZZ -X POST -d 'FUZZ=key' -H 'Content-Type: application/x-www-form-urlencoded' -ic -sa -mc all -fc 404 -o param_post_fuzz.txt` | Parameter Fuzzing - POST |
| `ffuf -u http://url.com/admin/admin.php -c -w ids.txt:FUZZ -X POST -d 'id=FUZZ' -H 'Content-Type: application/x-www-form-urlencoded' -ic -sa -mc all -fc 404 -o value_fuzz.txt`                                                                  | Value Fuzzing            |


>**ℹ️ Info:**
>
> When we find extensions, make a file containing them for further fuzzing later on.
>
>If we find a PHP file that tells us we're not authenticated or something similar
>we can use parameter fuzzing to try and see if there is something. If we have
>to make a post request to a PHP file it accepts `Content-Type: application/x-www-form-urlencoded`.
